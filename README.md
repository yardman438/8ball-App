# 8 BALL APP

Application for solving complex Yes / No questions.
Ask a question -> shake the device -> get an answer

## Project details:

- MVVM three-layers architecture with active model.
- Used CoreData.
- Used SnapKit,
- Used RxSwift,
- Used SwiftLint,
- Used SwiftGen.
